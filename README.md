# SMM Pro

Türkiye'de serbest meslek hekimleri için muayenehane muhasebe takip uygulaması.

## Özellikler
- Serbest Meslek Makbuzu (SMM) kayıtları
- Gider ve KDV takibi
- Aylık ve yıllık raporlar (Excel/PDF export)
- Belge fotoğrafı yükleme (Firebase Storage)
- 3 cihazda gerçek zamanlı sync (Firebase Realtime Database)
- PWA: offline çalışır, telefon/tablet/masaüstüne yüklenebilir

## Teknoloji
Vanilla JS + Firebase + GitHub Pages
